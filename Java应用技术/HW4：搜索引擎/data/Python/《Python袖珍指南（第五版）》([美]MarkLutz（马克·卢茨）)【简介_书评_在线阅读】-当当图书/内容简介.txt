Mark Lutz是Python培训的领军人物，最早且最畅销的Python书籍的作者和Python社区的先驱之一。 Mark是广受欢迎的O’Reilly书籍《Learning Python》、《Programming Python》和《Python Pocket Reference》的作者，三本书现在均已更新。他从1992年开始一直在使用和宣传Python，从1995开始撰写Python书籍，从1997年开始教授Python课程。到2010初，他已经教授了约250次Python培训课程，超过3000名学生，所撰写的Python书籍已经售出约25万本，并且翻译成数十种语言。 此外，他拥有威斯康辛大学的计算机科学学士与硕士学位，并且在近25年来作为一名职业开发者，在编译器、编程工具、应用程序辨析和多种客户端/服务器系统等领域工作。
