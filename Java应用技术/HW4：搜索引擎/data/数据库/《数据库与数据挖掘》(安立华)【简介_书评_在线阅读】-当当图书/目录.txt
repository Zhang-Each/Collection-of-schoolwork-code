前言
随着计算机应用技术和网络技术的普及，数据量急剧增加，人们借助计算机和数据库技术科学地保存和管理大量的、复杂的数据，以便能够更加方便、快捷地利用好这些珍贵的信息资源。在数据管理中，数据库系统起到了关键的作用，没有数据库的支持，根本不可能恰当地管理使用者需要的信息，数据库技术已经成为当今计算机信息系统的基础和核心。但是如何从海量的、分散的、复杂的数据海洋中，迅速找到有价值的和潜在有用的信息与知识，如何实现对这些数据的集中的组织、分析与管理，如何正确、有效地为企业做出决策，就需要数据仓库与数据挖掘来提供帮助了。数据挖掘理论与方法的研究与创新已经成为信息科学与管理领域最重要的课题之一。
本教材正是结合数据库与数据挖掘技术的课程特点及应用需求、本科生乃至研究生的特点而编写的，力求克服原理与应用分离的缺点，弱化理论知识的抽象，体现注重能力、内容创新、结构合理、叙述通俗、易学易用的特点，全书以图文并茂的形式介绍数据库与数据挖掘的理论知识，以Microsoft SQL Server 2008乃至更高版本为平台，详细介绍了如何利用Microsoft SQL Server 2008进行数据库操作和数据挖掘的知识和方法。
本书是课程组结合多年的教学经验编写的。主要包括10个章节，各章内容如下。
第1章数据库概论，介绍数据库的基本概念及理论，并详细介绍关系数据库的相关内容。
第2章SQL Server 2008数据库基础，详细介绍数据库、数据表、索引、视图等的建立与管理方法。
第3章数据库设计，介绍数据库设计步骤及方法。
第4章事务管理与并发控制，介绍事务的概念、特性及事务处理方式。
第5章数据库安全性与权限管理，介绍数据库安全措施及权限管理方式。
第6章数据仓库技术，介绍数据仓库的基本概念、体系结构及建立过程等。
第7章数据挖掘概述，介绍数据挖掘的相关概念及技术与工具。
第8章关联分析，介绍关联规则的基本概念与算法，详细介绍Apriori算法。
第9章聚类分析，介绍聚类的相关概念及算法，详细介绍Kmeans聚类法、Kmedoids聚类法。
第10章分类，介绍分类的相关概念及常用的算法，如决策树分类、贝叶斯分类等。
本书由东北林业大学工程技术学院安立华老师和鞍山师范学院娄雅斌教授主编，东北林业大学信息学院彭涛老师副主编，第1章、第2章由彭涛老师编写，第3～10章由安立华老师编写，娄雅斌教授负责全书的内容组织及校对。
本书在编写过程中查阅了大量的国内外文献资料，谨向参考文献中提到的各位学者表示感谢。由于疏忽而没有列出的文献，在此向涉及的各位作者深表歉意。同时，由于时间仓促和编者能力有限，书中难免存在一些不当之处，敬请广大读者批评指正。
