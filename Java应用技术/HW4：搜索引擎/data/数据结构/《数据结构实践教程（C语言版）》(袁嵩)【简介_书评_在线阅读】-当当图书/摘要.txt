本书是《数据结构》（C语言版）配套的实验指导、习题、课程设计，本教材内的所有语言采用C/C 语言，所有应用程序均在Microsoft Visual Studio 2010集成开发环境下编译并通过。本教材结合企业常用的实际案例、应用环境和软件开发技术，对线性表、队列、二叉树、图、查找等多个重要的数据结构知识单元进行了分析和设计，并为每个知识单元设计了实验项目和丰富的习题，力求使学生充分理解这些知识并通过实验、习题和综合实践来验证所学的知识和算法。
全书分为三大部分：*部分为实验指导，共安排了五个实验项目，可对应课堂练习或实验教学，，重点训练每一知识单元编程实现技巧；第二部分为习题，可对应课堂练习或课外作业，进一步巩固和检验各单元知识点掌握情况；第三部分为综合实践，可对应课程设计或综合实践，该部分利用一个游戏项目综合训练数据结构核心知识和算法在企业实际项目中的应用。
本教材适合各类高等院校计算机专业学生作为《数据结构》实践教学参考用书，也适合作为对数据结构与算法应用开发感兴趣的学习者的指导用书或参考书。
