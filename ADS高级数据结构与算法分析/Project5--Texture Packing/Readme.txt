File 'source code' stores the source code for this project, file 'document' stores the report for this project, while file 'test' stores the testing files for this project.
Using instruction 'cd source_code' and 'make' to generate executable file.
Using instruction 'cd test' and 'make -f generate.mk' to generate executable file for generating testing cases.
Using instruction 'cd test' and 'make -f texture_packing.mk' to generate executable file for checking testing cases.
